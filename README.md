## Get Random Verse
Random verso of Bible on English (NIV Version) and Indonesia (tb/bis)

- ### URL

  `/bible/random`

- ### Method

  `GET`

- ### URL Query

  `ver = 'tb' (default) | 'bis'`

- ### Success Response

  - #### Code: 200
  - #### Content:
